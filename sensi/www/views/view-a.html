Welcome to Sensi, your friendly super-simple Single Page App framework *

Overview

This framework presents a way of doing SPA development by using jQuery as the "backend base" for DOM referencing and framework work-horsing ( such as html view loads ) alongside Vue's "frontend base" for templating, data injection, app functions and event handling ( and whatever else you want to do with Vue ). Furthermore it is built on top of Cordova for app deployment, however you can run it directly in Firefox or on a localhost for ease and speed of development. To deploy and run in Cordova for your device, please follow the guide here: https://cordova.apache.org

Inspiration

Sensi is inspired by the classic Sensible Soccer video game, because we play it all the time for its purity and simplicity combined with just very, very good gameplay! Pick up and Play at its best. Hopefully this framework provides a similar kind of experience **.

Default UI

Sensi comes pre-packaged with a traditional "View Controller" style app header (like an iOS navigation controller ) & footer ( like an iOS tab bar ).

Sensi creates & appends views to the DOM and keeps them in memory as you request subviews, as you go back, these subviews are then destroyed ***.

This is all done with a standard sliding page transition - see how this works by clicking NEXT below and inspecting the DOM in the page source.

Views (like this one) are all standalone HTML files and contained within the "views" folder, however as long as they follow the same pattern you can put them where ever you like.
